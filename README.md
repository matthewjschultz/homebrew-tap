# homebrew-tap

Homebrew tap for macOS apps by Matthew Schultz.

## Install

```
brew tap matthewjschultz/tap
brew install claude-status
brew install miditransporter
brew install mjsfx
```

> **Homebrew 6+ tap trust:** if you have `HOMEBREW_REQUIRE_TAP_TRUST` set, trust the tap once before installing:
>
> ```
> brew trust --tap matthewjschultz/tap
> ```

## Apps

| Cask | Description |
|------|-------------|
| `claude-status` | Menu bar app monitoring Claude service health — [claudestatus.app](https://claudestatus.app) |
| `miditransporter` | Floating panel for sending MIDI transport messages — [miditransporter.app](https://miditransporter.app) |
| `mjsfx` | Native macOS designer for 8-bit retro game SFX — [mjsfx.app](https://mjsfx.app) |
| `claude-api-status` | Predecessor to Claude Status (EOL) |
