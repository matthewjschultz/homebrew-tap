# Security Policy

## Supported versions

Only the current `main` branch is supported; there are no maintained older versions.

## Reporting a vulnerability

This repo is a Homebrew tap (formula definitions only) — it has no build step and runs no
code of its own beyond what Homebrew's formula DSL executes to fetch and install the tapped
apps. If you find a vulnerability here (e.g. a formula pointing at a tampered download or
checksum), please report it privately using GitHub's **Report a vulnerability** button on
this repo's
[Security tab](https://github.com/matthewjschultz/homebrew-tap/security/advisories/new).

Please do not open a public issue for security reports.
